# United States — OSHA 29 CFR 1926 (construction standards / Focus Four map)

**Fragment ID:** `KB-REG-OSHA1926`
**What this is:** a copyright-safe **hazard → standard → control map** for OSHA's
construction standards (29 CFR 1926), anchored on the construction "Focus Four"
fatal-hazard groups (falls, struck-by, caught-in/between, electrocution) plus the
key program subparts a US construction site relies on.
**What this is NOT:** a reproduction of 29 CFR 1926 text. Cite the CFR
part/subpart numbers and the duty topics only — never paste the regulation
wording.

> Source: OSHA 29 CFR 1926 (construction) + the OSHA construction "Focus Four" framing — citation map · Year: 2026 · Reviewed: 2026-06-21 · Volatile: false (stable CFR structure; refresh on review).

29 CFR 1926 governs construction work under the OSH Act in the United States. It
grounds US construction permit/plan, JSA and risk-assessment skills with the
standard reference and the control expectation per hazard family, applied through
the hierarchy of controls.

## Hazard / topic → standard → control it grounds

| Hazard / topic | Standard (subpart) | Control expectation it grounds |
|---|---|---|
| Falls from height | 1926 Subpart M | fall-protection plan (guardrail / PFAS / hole covers) above the trigger height |
| Struck-by (vehicles, objects, equipment) | 1926 Subparts O / Q / R | traffic/exclusion-zone + overhead-protection controls |
| Caught-in / between (excavations, equipment) | 1926 Subpart P | protective system (slope / bench / shore / shield) per soil class |
| Electrocution (contact, energized work) | 1926 Subpart K | energized-work controls + LOTO + clearance distances |
| Scaffolds | 1926 Subpart L | scaffold design / inspection / access |
| Cranes & derricks | 1926 Subpart CC | lift plan + operator qualification |
| Hazard communication | 1926.59 → 1910.1200 | HazCom program + SDS access + labelling |
| Recordkeeping / reporting | 29 CFR 1904 | injury/illness log; fatality/hospitalization reporting |

## How the skill uses this fragment
- A US construction skill resolves the task's hazard family first, then cites the
  governing subpart and grounds the expected control through the hierarchy of
  controls (no PPE-only treatment without justification).
- Trigger heights / soil classes / clearance distances are stated as the *test*;
  the site facts decide. A missing input is recorded as `[GAP]`, never invented.
