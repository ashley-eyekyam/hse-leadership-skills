# Pre-output Quality Checklist — mine-incident-investigation

- [ ] De-identification ran FIRST; no injured/deceased-miner identifier, no exact pit/shaft location, no <5 fatality/injury cell, no re-identification key in the output.
- [ ] Every cause is evidence-backed (cites an E-n item); the RCA reaches a systemic/organisational factor (ICAM) — **not 'miner error'**.
- [ ] DGMS reportability is region-resolved; the **24h notice + Form J** are cited as verified anchors; any other DGMS form id is `[GAP]` — **no invented form id**.
- [ ] Every CAPA traces to a named cause with owner + due date; HoC-tagged; no PPE/admin-only without justification.
- [ ] `incident_rates` used as context only (no headline rate without hours+counts).
- [ ] The output reads as a draft for a competent (DGMS-qualified) person's review — never a sign-off.
