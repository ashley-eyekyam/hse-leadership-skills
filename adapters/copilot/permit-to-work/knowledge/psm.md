# OSHA PSM — Process Safety Management 14-element framework

**Fragment ID:** `KB-STD-PSM`
**What this is:** a copyright-safe **clause → output-artifact map** — the 14 PSM
element *numbers/topics* (OSHA 29 CFR 1910.119) plus the artifact each element
produces and the skill it grounds.
**What this is NOT:** a reproduction of the standard's normative text. 29 CFR
1910.119 is publicly available; cite the paragraph numbers and topics only —
never paste the regulation's wording. The US-statutory citation hook lives on
`KB-REG-US-OSHA`; this fragment is the jurisdiction-independent 14-element
*structure* (usable as a framework anywhere).

> Source: OSHA 29 CFR 1910.119 — element structure (framework reference; user holds the standard) · Year: 2026 · Reviewed: 2026-05-15 · Volatile: false (the 14-element structure is stable; the US-statutory citation lives on KB-REG-US-OSHA).

PSM is the process-safety backbone of the pack. The 14 elements form an audit
spine: `psm-program-manager` tracks per-element status; `management-of-change`,
`mechanical-integrity`, and the PHA facilitators each implement one or more
elements.

## How skills use this fragment
- `psm-program-manager` builds the 14-element status matrix; each element is an
  audit criterion (compliant / gap / overdue) with an owner + evidence.
- The PHA facilitators ground element (e); `management-of-change` grounds (l)+(i);
  `mechanical-integrity` grounds (j)+(d).
