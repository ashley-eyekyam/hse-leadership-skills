# BS EN 15635:2008 + SEMA Code of Practice — racking application & maintenance map

**Fragment ID:** `KB-STD-EN15635-SEMA`
**What this is:** a copyright-safe **inspection-regime + RAG-action map** for the
operation, inspection and maintenance of steel static storage (pallet racking) —
grounding `warehouse-racking-mhe-safety` (LOG-02). It names the PRRS role, the
inspection cadence, and the SEMA Red/Amber/Green damage-action ranking.
**What this is NOT:** a reproduction of BS EN 15635:2008 or the SEMA Code of
Practice text. Cite the standard number and the inspection/RAG topics only —
never paste the standard or code wording.

> Source: BS EN 15635:2008 (Steel static storage systems — Application and maintenance of storage equipment) + SEMA Code of Practice — citation map · Year: 2026 · Reviewed: 2026-06-23 · Volatile: false.

BS EN 15635:2008 sets the duties of the **specifier/supplier/user** for in-service
racking, and the SEMA Code of Practice gives the UK industry application. Together
they ground a warehouse racking-safety artifact with the inspection regime and the
damage-classification action map a defensible racking inspection must produce.

## Inspection regime → duty → artifact it grounds

| Element | Duty (paraphrased) | Artifact it grounds |
|---|---|---|
| **PRRS** (Person Responsible for Racking Safety) | A named, competent duty-holder owns racking safety, damage reporting and the inspection regime | named PRRS appointment + responsibility record |
| Weekly / regular visual inspection | A trained worker carries out routine visual checks at an interval set by the site risk assessment | routine inspection log |
| Formal "expert" inspection **≥ every 12 months** | A technically-competent person carries out a documented expert inspection at least annually (more often — e.g. 6-monthly — for high-traffic / 24-7 sites) | annual expert inspection report |
| Damage report & immediate-isolation route | Any worker can report damage; serious damage triggers immediate isolation/off-load | damage report + isolation record |
| SWL load notices | Maximum permissible load notices displayed; configuration not changed without recalculation | SWL load-notice display |

## SEMA Red/Amber/Green (RAG) damage-action ranking

The SEMA traffic-light system **classifies damage and sets the action** — it is a
*damage/severity classification*, not a replacement for the 5×5 residual risk score:

| Band | Meaning | Action |
|---|---|---|
| **Green** | Within tolerance | Log and continue to monitor at the routine interval |
| **Amber** | Exceeds tolerance | Repair/replace within 4 weeks; if not actioned it **auto-escalates to Red** (the off-load nuance — whether Amber also requires immediate off-load — is confirmed by the competent person / SEMA-approved inspector for the site) |
| **Red** | Critical | **Immediately off-load and isolate** the affected run; repair before reuse |

Tolerance reference: the max permissible deformation is on the order of the
manufacturer's stated tolerance (commonly described as up to ~2× a stated limit
for the boundary between bands); the **site-specific tolerance values, SWL ratings
and bay configurations are `[GAP]`** — never assume an SWL.

## How the skill uses this fragment
- LOG-02 resolves the PRRS + inspection cadence first, then classifies any damage
  through the SEMA RAG band and grounds the matching action (Green=monitor /
  Amber=4-week repair+auto-escalate / Red=immediate off-load+isolate).
- The hierarchy of controls applies at design/inspection (engineered protection +
  inspection regime) before relying on signage/PPE; see `KB-SNIP-RACKING-MHE`.
- Any unstated SWL, tolerance or inspection interval is `[GAP]`, never invented.
