# NFPA 652 / NFPA 660 — combustible-dust DHA + parameters (clause→artifact)

**Fragment IDs:** `KB-STD-NFPA-652`, `KB-STD-NFPA-660`
**What this is:** a copyright-safe **DHA-element → artifact map** for NFPA 652
(Fundamentals of Combustible Dust) and NFPA 660 (the consolidated combustible-dusts
standard that draws together 652/654/61/484/655), plus the **named** dust-explosion
parameter set the assessment resolves (with source+year), never invented values.
**What this is NOT:** a reproduction of the NFPA normative text or a table of
remembered Kst / Pmax / MIE / MIT figures. The user holds the licensed NFPA standard and resolves the
binding dust test data from a competent laboratory; cite the section structure only.

> Source: NFPA 652 (Fundamentals of Combustible Dust) + NFPA 660 (consolidated combustible dusts) — element structure (user holds the standard) · Year: 2026 · Reviewed: 2026-05-15 · Volatile: false (multi-year revision cycle).

`KB-STD-NFPA-660` **supersedes** `KB-STD-NFPA-652` (the consolidation; recorded in the
registry `supersedes` field so existing DHAs that cite 652 still resolve — the
A3 legacy-preservation pattern). New assessments cite 660; legacy ones keep 652.

## Dust-explosion parameter set (named; resolved with source+year, NEVER invented)

| Parameter | Meaning | Discipline |
|---|---|---|
| Kst | deflagration index (bar·m/s) → dust explosion class St-1/2/3 | resolved from a lab test report (cite lab + year); `[GAP]` if untested |
| Pmax | maximum explosion pressure (bar) | lab-resolved; `[GAP]` if untested |
| MIE | minimum ignition energy (mJ) | lab-resolved; drives ignition-source control |
| MIT | minimum ignition temperature (°C) | lab-resolved; drives surface-temperature limits |

**Never quote a remembered Kst/Pmax/MIE/MIT for a named dust** — resolve it from the
user's lab test data with the source+year, or `[GAP]`-flag it and route to testing.
The citation grader fails a fabricated dust parameter.

## How the skills use these fragments
- `reactive-dust-explosion-assessment` runs the DHA element-by-element, resolving the
  dust parameters (or `[GAP]`) and grounding the DSEAR basis of safety + ATEX zone
  (`dsear-atex.md`); the deflagration/explosion scenarios are handed to
  `hazop-facilitator` for the structured study.
