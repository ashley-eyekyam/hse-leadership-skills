# Pre-output Quality Checklist — traffic-management-plan

Before producing any output, validate the draft against this gate. A failure is a defect to fix,
not a note to ship.

## The named-site GATE (refuse-on-vague)
- [ ] The **site is named** with its **access points** (Q1) — not a generic site; an unnamed
  access point or "the site roads" is a `[GAP]` and a **stop**.
- [ ] **At least one vehicle type AND pedestrians / operatives** (Q2) are present — the interface
  is the whole point; no plan ships on a generic site or an invented layout.
- [ ] The **routes are named** (Q1 / Q3) — no plan is produced on "the site roads"; the conflict
  points are named, never invented.

## Segregation-by-design (the core value)
- [ ] **Every** vehicle-pedestrian conflict point is treated by the **segregation-by-design
  hierarchy** (`KB-SNIP-TRAFFIC-SEGREGATION`): **eliminate / design out reversing → one-way &
  turning → physical segregation → signage / speed / lighting → banksman LAST**.
- [ ] **No un-justified "hi-vis + banksman only" pedestrian control** ships — a pedestrian control
  with no physical or temporal segregation is flagged PPE/admin-led and **pushed up the
  hierarchy**, never the headline control.
- [ ] **Reversing leads with elimination** (one-way / turning circle / drive-through bay); the
  banksman is the residual control after the higher-order options are designed in.
- [ ] **Routes are suitable, sufficient and separated** (Schedule 3) with **warning of approach**.

## Delivery, signage & the public interface
- [ ] **Loading / delivery is managed** (booked-in windows, bays, holding area, routing) to remove
  ad-hoc reversing; **speed limits, signage, and lighting** are set at conflict points.
- [ ] The **public / live-highway interface** is protected (hoarding / covered way / protected
  footway + the highway-signing duty) — **the public is never controlled by a banksman alone**.

## Defensibility & citation
- [ ] Every enforcement / monitoring / review action carries a **named owner (role) + a review
  date + a trigger** (no anonymous actions, no "ASAP").
- [ ] Every citation traces to the KB — **CDM 2015 Reg 27 + Schedule 3** + HSG144 (UK), **29 CFR
  1926 Subpart O** (US); for **India** the **state is resolved** (defers to `hse-india`) and a
  **literal `[GAP]`** is emitted, **never a national form number**. **Uncontrolled reversing with
  no Reg 27 / Schedule 3 grounding is a citation + specificity failure.**

## De-identification
- [ ] De-id pass complete **BEFORE drafting**; the TMP is **site / route-level and carries no named
  individual** — any named driver / operative / banksman from the inputs is scrubbed to a role /
  location label; **no fitness-for-duty / health detail** is circulated; no embedded re-id key; no
  unsuppressed `<5` injury cell.

## SME boundary
- [ ] The output **never** claims "approved by a competent person" — it is decision-support that
  precedes the human competent-person review.
