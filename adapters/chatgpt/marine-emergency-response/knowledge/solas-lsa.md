# SOLAS Chapter III + LSA Code — life-saving appliances & muster duty map

**Fragment ID:** `KB-REG-SOLAS-LSA`
**What this is:** a copyright-safe **appliance → muster → competence map** for
life-saving appliances and emergency mustering — grounding
`marine-emergency-response` (MAR-03). It names survival craft (TEMPSC), the
capacity-vs-POB check, the station bill and the OPITO competence requirement.
**What this is NOT:** a reproduction of SOLAS, the LSA Code or OPITO module text —
OPITO BOSIET/HUET curriculum is a **licensed competence scheme → cite the

SOLAS Chapter III and the LSA Code set the requirements for survival craft,
muster and drills for vessels and (by application) offshore installations. They
ground the survival-craft + station-bill limb of a marine emergency artifact.

## Element → duty → artifact it grounds

| Element | Duty (paraphrased) | Artifact it grounds |
|---|---|---|
| Survival craft (**TEMPSC** — totally-enclosed lifeboats; davit/free-fall launch) | Provide survival craft of a type and number appropriate to the installation/vessel | survival-craft provision basis |
| Capacity vs POB | Survival-craft capacity must cover the persons-on-board | capacity-vs-POB check |
| **Station bill / muster list** | Assign every person an emergency duty and a muster station | station bill (role-labelled) |
| Drills | Conduct regular abandonment/emergency drills | drill schedule + records |
| **OPITO BOSIET/HUET** competence | Personnel hold the required offshore survival/escape competence | competence requirement (cert detail `[GAP]`) |

## Discipline
- The station bill assigns **roles** — in any artifact, persons are **role-labelled,
  never named** (de-identify first).
- Vessel/installation-specific survival-craft capacity figures are `[GAP]` until
  supplied; never assumed.
- OPITO certificate numbers / module curriculum are licensed → cite the competence
  *requirement* only; certificate detail `[GAP]`.

## How the skill uses this fragment
MAR-03 grounds survival-craft provision, capacity-vs-POB, the role-labelled station
bill and the OPITO competence requirement; pairs with `KB-REG-PFEER` for the
muster/TR/EER chain.
