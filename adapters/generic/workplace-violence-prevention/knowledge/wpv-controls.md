# Workplace-violence controls — environmental & administrative before reactive

**Fragment ID:** `KB-SNIP-WPV-CONTROLS`
**This is prompt text, applied by the model — not a generator.** It is the

## Instruction — the WPV control order

1. **Worksite analysis (first).** Use records review, the incident log, walkthroughs,
   and staff input to identify WPV hazards and high-risk areas/tasks (type-1
   criminal, type-2 client/patient, type-3 worker-on-worker, type-4 personal-relationship).
2. **Environmental / engineering controls.** Sightlines and lighting, controlled
   access and exits, barriers/enclosures at reception/triage, alarm and
   communication systems, secure design of high-risk areas.
3. **Administrative / work-practice controls.** Staffing and skill mix, flagging of
   known-risk patients (de-identified), visitor management, work-alone policy,
   reporting and post-incident support, and a no-retaliation reporting culture.
4. **Training + response (residual).** De-escalation and response training, response
   teams, and personal alarms as the residual line — never the headline control.

## The gate (reject these)
- A plan that leads with training/response and omits environmental and
  administrative controls → **reject** (reactive-led).
- A control set not grounded in a worksite analysis → **reject** as not defensible.
- Reporting incident counts in a way that allows re-identification of a worker,
  patient, or unit → **reject** (apply small-cell suppression).
