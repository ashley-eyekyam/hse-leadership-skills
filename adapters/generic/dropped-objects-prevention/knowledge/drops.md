# Dropped objects — DROPS Recommended Practice + IADC + API duty map

**Fragment ID:** `KB-REG-DROPS`
**What this is:** a copyright-safe **survey → taxonomy → securing-hierarchy map**
for dropped-object prevention on offshore installations — grounding
`dropped-objects-prevention` (MAR-02). It names the DROPS survey, the static vs
dynamic taxonomy, the reliable-securing hierarchy and exclusion zones.
**What this is NOT:** a reproduction of the DROPS Recommended Practice, IADC HSE
Guidelines or API RP text — and **the DROPS Calculator consequence-band table is
licensed → cite-only** (see `KB-DATA-DROPS-IMPACT`). Cite the practice/standard
numbers and the topics only.

> Source: DROPS Recommended Practice (2017) + DROPS Reliable Securing + IADC HSE Guidelines Section 16 (Dropped Objects) + API RP 2D (offshore crane O&M) / API RP 54 (drilling & servicing) — citation map · Year: 2026 · Reviewed: 2026-06-23 · Volatile: true.

DROPS (Dropped Objects Prevention Scheme) gives the recommended practice for
identifying, surveying and securing potential dropped objects offshore; IADC
Section 16 and the API RPs give the drilling/crane application. Together they
ground a dropped-objects prevention artifact.

## Element → duty → artifact it grounds

| Element | Duty (paraphrased) | Artifact it grounds |
|---|---|---|
| **DROPS survey** | Systematically survey fixed and potential dropped objects (derrick, at-height structures, equipment) | dropped-object survey + register |
| **Static vs dynamic taxonomy** | Classify a dropped object as static (falls from a static position) or dynamic (knocked/swung/dropped during an operation) | object taxonomy in the register |
| **Reliable securing hierarchy** | Secure at-height items with primary fixings + secondary retention (tethering/lanyards) to a reliable-securing standard | securing/inspection plan |
| **Exclusion zones** | Establish red/exclusion zones below at-height work so people are not under suspended/at-height loads | exclusion-zone control |
| API RP 2D / RP 54 | Crane and drilling/servicing operation & maintenance controls | crane/drilling securing controls |

## The control order (securing/survey/exclusion first)
Dropped-object risk is controlled by **survey + reliable securing + exclusion
zones first** — "hard hats for people below" is a residual control, never the
headline (see `KB-SNIP-DROPS-SECURING`).
