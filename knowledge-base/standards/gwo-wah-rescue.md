# GWO Working at Heights / First Aid / ART — wind competence & rescue map

**Fragment ID:** `KB-STD-GWO-WAH-RESCUE`
**What this is:** a copyright-safe **competence-requirement + rescue-discipline map**
for wind-turbine work at height — grounding `wind-turbine-work-at-height-rescue`
(REN-01). It names the GWO competence requirement (and refresh cadence), the
two-person climb-team discipline and the "emergency services alone is not a rescue
plan" rule.
**What this is NOT:** a reproduction of the GWO module curriculum or certificate
detail — the **GWO module text and certificate numbers are a licensed competence
scheme → cite the competence *requirement* only**; certificate detail `[GAP]`.

> Source: Global Wind Organisation (GWO) Basic Safety Training — Working at Heights module + GWO First Aid + GWO Advanced Rescue Training (ART) (cite competence requirement only; module curriculum/cert detail licensed) · Year: 2026 · Reviewed: 2026-06-23 · Volatile: false.

GWO is the wind-industry competence scheme for working at height, first aid and
advanced rescue. It grounds the competence + rescue-discipline limb of a wind WAH
artifact (alongside the statutory `KB-REG-WAH2005`).

## Element → requirement → artifact it grounds

| Element | Requirement (paraphrased) | Artifact it grounds |
|---|---|---|
| GWO Working at Heights competence | Workers at height in wind hold current GWO WAH competence | competence requirement record |
| Competence refresh | GWO competence is **refreshed every 2 years** (24-month validity) | refresh-due tracking |
| GWO First Aid / ART | First-aid and advanced-rescue competence for the rescue team | rescue-team competence |
| Two-person-minimum climb team | A climb is a **two-person-minimum** team with ground support | climb-team manning |
| Suspended-worker recovery | A suspended worker is recovered **within minutes** (suspension trauma) | timed rescue capability |

## The rescue discipline
- **"Emergency services alone is NOT a rescue plan."** The team's own tested rescue
  capability recovers a casualty in minutes; calling 999/112 is a supplement.
- Certificate numbers / module curriculum are **licensed** → cite the competence
  *requirement* only; the worker's actual certificate detail is `[GAP]`.

## How the skill uses this fragment
REN-01 grounds the climb-team competence + the timed, team-owned rescue capability
on this map (with `KB-SNIP-RESCUE-PLAN`), and records the GWO competence requirement
without reproducing the licensed module/certificate detail (`[GAP]`).
