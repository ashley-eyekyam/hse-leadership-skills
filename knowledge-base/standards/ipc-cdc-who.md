# Infection prevention & control — CDC isolation precautions + WHO IPC core components + Spaulding

**Fragment ID:** `KB-STD-IPC-CDC-WHO`
**What this is:** a copyright-safe **topic → artifact** structure map for the main
infection-prevention-and-control (IPC) reference frameworks — the CDC Guideline
for Isolation Precautions (Standard + Transmission-Based precautions), the WHO
core components of IPC programmes, and the Spaulding classification for
device reprocessing. It grounds an IPC-plan skill in the precautions-by-route
logic and the IPC programme structure.
**What this is NOT:** a reproduction of the guideline text. Cite the precaution
categories, the core-component topics, and the Spaulding levels only — never paste
the wording (user holds the licensed/public source).

| Topic (cite, don't paste) | Focus | Artifact it grounds |

## How the skill uses this fragment
- Grounds an infection-control-plan skill in Standard + Transmission-Based
  precautions selected by route (`KB-SNIP-IPC-PRECAUTIONS`), the WHO core-component
  programme structure, and the Spaulding reprocessing decision.
- The hierarchy of controls applies: eliminate/engineer (ventilation, single
  rooms, safer devices) and administrative controls before relying on PPE.
- Any case/cluster detail is **de-identified** per the mandatory de-id block + the
  PHI extension; small clusters are suppressed against re-identification.
