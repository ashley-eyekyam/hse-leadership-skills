## Level-crossing remedial hierarchy (highest order first)

1. **Closure** — eliminate the crossing.
2. **Grade separation** — bridge/underpass so road and rail do not cross at level.
3. **Engineering** — barriers / obstacle detection / MSL / full-barrier protection.
4. **Sighting / signage / administrative** — last, where a higher order is not
   reasonably practicable.

## Track-worker protection hierarchy (highest order first)

1. **Separation** — green-zone working / line blockage / possession (work with no
   trains running).
2. **Safe systems of work (SSOW)** — planned, authorised system for the task.
3. **Warning** — TOWS (Train Operated Warning System) / other warning systems.
4. **Lookout-only** — last resort, where no higher protection is reasonably
   practicable.

Competence: the **COSS** (Controller of Site Safety) role and **Sentinel**
competence framing apply; red-zone (lookout-only) working is minimised.
