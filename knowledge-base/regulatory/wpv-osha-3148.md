# United States — OSHA 3148 workplace-violence prevention program map (healthcare)

**Fragment ID:** `KB-REG-WPV-OSHA3148`
**What this is:** a copyright-safe **program-element map** for OSHA Publication
3148 (Guidelines for Preventing Workplace Violence for Healthcare and Social
Service Workers) — the five program elements a workplace-violence (WPV)
prevention program should contain, framed as duty → artifact for a healthcare or
social-service employer. Enforcement leans on the OSH Act General Duty Clause
(Section 5(a)(1)) where a recognized WPV hazard exists.
**What this is NOT:** a reproduction of OSHA 3148 text. Cite the program-element
topics and the General Duty Clause only — never paste the guideline wording.

> Source: OSHA Publication 3148 (Workplace Violence Prevention for Healthcare & Social Service) + OSH Act §5(a)(1) General Duty Clause — citation map · Year: 2026 · Reviewed: 2026-06-21 · Volatile: true (guidance + any future WPV standard drift; refresh on review).

OSHA 3148 is *guidance* (not a stand-alone standard); the enforceable hook is the
General Duty Clause where WPV is a recognized hazard. This fragment grounds a
healthcare WPV-program skill with the five-element structure a defensible program
must produce.

## Program element → duty → artifact it grounds

| Program element | Duty (paraphrased) | Artifact it grounds |
|---|---|---|
| Management commitment & worker participation | Assign responsibility; involve workers | WPV policy + program-ownership record |
| Worksite analysis & hazard identification | Identify WPV hazards (records review, walkthrough, incident log) | WPV hazard assessment |
| Hazard prevention & control | Apply engineering / administrative / behavioural controls | WPV control plan (hierarchy of controls) |
| Safety & health training | Train staff on recognition, de-escalation, reporting | training records |
| Recordkeeping & program evaluation | Log incidents; review and improve the program | WPV incident log + annual program review |

## How the skill uses this fragment
- A healthcare WPV skill grounds each of the five elements above and applies the
  hierarchy of controls at the prevention/control step (engineering and
  administrative controls before relying on response/PPE).
- The General Duty Clause is cited as the enforcement basis where a recognized WPV
  hazard exists. De-identify any incident detail per the mandatory de-id block; a
  missing element is `[GAP]`, never invented.
