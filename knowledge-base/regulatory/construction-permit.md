# Cross-list — Construction high-risk-activity permit-to-work map

**Fragment ID:** `KB-REG-CONSTRUCTION-PERMIT`
**What this is:** a copyright-safe **activity → permit → control map** for the
construction high-risk activities that require a permit-to-work (PTW) before
starting — the per-facet content the construction sector pack cross-lists onto its
permit/JSA skills. It names the activity classes, the permit instrument, and the
pre-start controls a permit must confirm, grounded jurisdiction-neutrally and
read alongside the jurisdiction map (`KB-REG-CDM2015` UK / `KB-REG-OSHA1926` US).
**What this is NOT:** a reproduction of any standard or permit template, and not a
jurisdiction statute map. Cite the activity/control topic only — never paste
standard wording.

> Source: construction permit-to-work practice (cross-walks KB-REG-CDM2015 / KB-REG-OSHA1926) — practice map · Year: 2026 · Reviewed: 2026-06-21 · Volatile: false (stable practice; refresh on review).

## High-risk activity → permit → control it grounds

| Activity | Permit instrument | Pre-start control it grounds |
|---|---|---|
| Excavation / ground disturbance | excavation permit | utility-scan + protective-system (slope/shore/shield) confirmation |
| Working at height | working-at-height permit | edge protection / fall-arrest + rescue plan |
| Hot work | hot-work permit | fire watch + atmosphere + combustible exclusion zone |
| Confined-space entry | confined-space entry permit | atmosphere test + ventilation + standby + rescue |
| Lifting operations | lift permit | lift plan + exclusion zone + appointed person |
| Energized / electrical work | electrical permit | isolation/LOTO (see KB-REG-LOTO) + clearance distance |

## How the skill uses this fragment
- A construction permit/JSA skill resolves the activity class first, grounds the
  permit instrument and the pre-start controls here, then cites the governing
  jurisdiction map for the statutory hook.
- Controls follow the hierarchy of controls; a missing input is `[GAP]`.
- `bundled_in` cross-list wiring to the sector pack is deferred to Phase 17
  (SEAM-01) — content only here.
