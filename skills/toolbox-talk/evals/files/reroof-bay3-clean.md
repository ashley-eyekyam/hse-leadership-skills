# Toolbox Talk — Working at Height: Re-roofing Bay 3 (specificity fixture, de-id clean)

A specific, frontline-ready toolbox talk for a named task. De-identification holds —
no personal names, no <5 injury cells, no re-identification key; any incident is a
clearly-labelled typical example, not a fabricated local event.

## Why this matters today

We are re-roofing Bay 3 this shift and the cherry-picker (MEWP) is out of service, so
the crew is working off the leading edge by hand. A fall from this roof is
life-changing or fatal. This five-minute talk keeps everyone off that edge unprotected.

## The specific hazards on this task

- Unprotected leading edge on the Bay 3 roof — fall from height (~6 m to the slab).
- The MEWP is out of service, so the planned safe access is unavailable.
- Mixed crew — differing familiarity with this roof profile and the edge-protection layout.
- Materials and tools at the edge — dropped-object risk to anyone below.

## Key controls (higher-order first)

- Elimination: pre-assemble roof sections at ground level and lift them as units where
  the task allows — owner: Site Supervisor.
- Engineering: install a temporary edge-protection guardrail along the Bay 3 working
  edge before anyone goes near it — owner: Scaffold Lead.
- Engineering: while the MEWP is down, restrict work to inside the guardrail; anchored
  harness + running line for any task that must reach the edge — owner: Working-at-Height Supervisor.
- Administrative: permit-to-work + a briefed exclusion zone below; tethered tools —
  owner: Supervisor.
- PPE (last line, never the only line): hard hats + inspected harnesses — owner: each crew member.

## Learn from this (illustrative / typical example — not a local incident)

On a similar re-roofing job elsewhere, a worker stepped back toward an unguarded edge
while carrying a sheet and was only stopped by a co-worker's shout. The lesson: edge
protection goes up BEFORE the work starts.

## Discussion prompts

- With the MEWP out of service, what is OUR safe way to reach the edge today?
- Where is the edge protection going, and who checks it before we start?
- If something feels wrong up there, what do you do and who do you tell?

## Takeaways

- No one approaches the Bay 3 edge until the guardrail is up and checked.
- MEWP is down — edge tasks are harness + running line, or they wait.
- Tools tethered, materials back from the edge, exclusion zone clear.
- See a hazard, stop the job and speak up.
