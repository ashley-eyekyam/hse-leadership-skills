# Pre-output Quality Checklist — dgms-statutory-pack

Before producing any output:

- [ ] The mine's **region/zone is resolved** before any DGMS form is cited (mandatory).
- [ ] Every cited DGMS form is one of the **five verified anchors** OR is recorded `[GAP]` — **no invented form id**.
- [ ] The OSH-Code transition note is appended (legacy-first).
- [ ] De-identification pass complete BEFORE drafting; injured/deceased-miner identity scrubbed to role labels; no <5 fatality/injury cell published; no re-identification key in the document.
- [ ] Every follow-up action has a named owner + due date (`smart_actions`).
- [ ] The output reads as a draft for a competent (DGMS-qualified) person's review — never as a sign-off.
